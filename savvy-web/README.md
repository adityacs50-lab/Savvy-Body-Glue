# SAVVY — Landing Page

Production-quality fashion e-commerce landing page for **SAVVY Body Glue**.

## Run locally

```bash
cd savvy-web
npm install
npm run dev
```

Open http://127.0.0.1:5173/

## Build

```bash
npm run build
npm run preview
```

## Brand assets

Real brand kit assets live in `public/assets/`:

- `brand/` — SAVVY logo
- `products/` — tube + pouch cutouts
- `illustrations/` — flower motif
- `editorial/` — Canva reference

Colors sampled from the brand kit:

- Dark green `#1C3628` / `#0A1510`
- Dusty pink `#D08A99`
- Highlight `#D9BFC5`
- Cream `#FFEDF1`

Fonts: League Spartan (brand), Instrument Serif (editorial), Syne (display accents).
