# SAVVY Body Glue

Premium Gen-Z fashion landing page for **SAVVY Body Glue**.

## Quick start

```bash
cd savvy-web
npm install
npm run dev
```

Open http://127.0.0.1:5173/

## Build

```bash
cd savvy-web
npm run build
npm run preview
```

## Stack

- React + TypeScript + Vite
- GSAP / ScrollTrigger
- Brand assets from the SAVVY kit (logo, tube, pouch, flower)

## Project layout

- `savvy-web/` — production website
- `Savvy brand kit/` — original brand source files
